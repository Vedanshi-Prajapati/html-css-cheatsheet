HTML TABLE

A table in HTML consists of table cells inside rows and columns. It uses <table> , <tr> (table rows) , <th> (table header) , <td> (table data). A table cell can contain all sorts of HTML elements: text, images, lists, links, other tables, etc.

HTML Table Tags

  1. <table> Defines a table
  2. <th> Defines a header cell in a table. By default, table headers are bold and centered.
  3. <tr> Defines a row in a table
  4. <td> Defines a cell in a table
  5. <caption> Defines a table caption
  6. <colgroup> Specifies a group of one or more columns in a table for formatting
  7. <col> Specifies column properties for each column within a element
  8. <thead> Groups the header content in a table
  9. <tbody> Groups the body content in a table
  10. <tfoot> Groups the footer content in a table
1. Table Borders
  • To add a border, use the CSS border property on table, th, and td elements
  • To avoid having double borders like in the example above, set the CSS border-collapse property to collapse.
  • Style table border using CSS properties like border-color, border-collapse, border-style, border-radius, background-color.
2. Table Sizes
Use the style attribute with the width or height properties to specify the size of a table, row or column.
3. Table Caption
  • You can add a caption that serves as a heading for the entire table.
  • The <caption> tag should be inserted immediately after the <table> tag.
  • To add a caption to a table, use the <caption> tag
4. Cell Padding and Spacing
  • Cell padding is the space between the cell edges and the cell content. Default value is 0px.
  • Cell spacing is the space between each cell. default value is 2px.
  • To add padding and spacing on table cells, use the CSS padding property on td, th.
5. Table Colspan & Rowspan
  • The value of the colspan and Rowspan attribute represents the number of columns and rows to span respectively.
