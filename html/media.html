HTML MEDIA

Video
  • <video> is used to show a video on a web page.
  • Attributes: controls, autoplay, muted, height, width
  • controls adds video controls, like play, pause, and volume.
  • The <source> element allows you to specify alternative video files which the browser may choose from.
  • <track> tag specifies text tracks for <audio> or <video> elements.
  • The text between <video> and </video> tags will only be displayed in browsers that do not support <video> element.
Audio
  • <audio> element is used to play an audio file on a web page.
  • Attributes: controls, autoplay, muted, height, width (same as <video>)
  • All same points as <video>.
Plug-ins
  • Plug-ins are computer programs that extend the standard functionality of the browser.
<object> Element
  • <object> element defines an embedded object within an HTML document.
  • It was designed to embed plug-ins (like Java applets, PDF readers, and Flash Players) in web pages, but can also be used to include HTML in HTML.
<embed> Element
  • Defines an embedded object within an HTML document.
  • <embed> element does not have a closing tag. It can not contain alternative text.
  • can also be used to include HTML in HTML
