BASIC TAGS

1. Headings
HTML headings are titles or subtitles that you want to display on a webpage. HTML headings are defined with the h1 to h6 tags from most to least important. Browsers automatically add some white space (a margin) before and after a heading. Use HTML headings for headings only. Don't use headings to make text BIG or bold. Don't use headings to make text BIG or bold.

Syntax: <h1> </h1> to <h6> </h6>

2. Paragraph
This element defines a paragraph that starts on a new line, and is usually a block of text. Browsers automatically add some white space (a margin) before and after a paragraph.

Syntax: <p> content </p>

3. Add White Space
<pre> content with spaces </pre> is used to defines preformatted text. The text inside it is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks.
<hr> is an empty tag used add horizontal rules.
<br> is an empty tag used add line breaks.
4. Style Attribute
Setting the style of an HTML element, can be done with the style attribute, using css property and its value.Some properties are: background-color, color(for text colour), font-family, font-size, text-align.

Syntax: <tagname style="property:value;">

6. Comment Tag
HTML comments are not displayed in the browser, but they can help document your HTML source code.
shortcut key: cmd + / or ctrl + /

Syntax: <!-- comment's content >
