IMAGES IN HTML

The HTML <img> tag is used to embed an image in a web page. It is an empty tag and has two required attributes.
Syntax: <img src="url" alt="alternatetext">
  1. the required src attribute specifies the path (URL) to the image.
  2. The required alt attribute provides an alternate text for an image
  3. Use the HTML width and height attributes or the CSS width and height properties to define the size of the image
  4. If width and height are not specified, the web page might flicker while the image loads.
  5. Use the CSS float property to let the image float to the left or to the right
  6. To use an image as a link, put the <img> tag inside the <a> tag.
  7. HTML allows animated GIFs.
