BLOCK AND INLINE ELEMENTS

Every HTML element has a default display value, depending on what type of element it is. The two most common display values are block and inline.
Block-level Elements
  • Always starts on a new line, browsers automatically add some space (a margin) before and after the element.
  • Stretches out to the left and right as far as it can
  • Two commonly used block elements are: <p> and <div>.
  • The <p> element defines a paragraph in an HTML document.
  • The <div> element defines a division or a section in an HTML document.
Inline Elements
  • Does not start on a new line.
  • Only takes up as much width as necessary.
  • Cannot contain a block-level element.
  • <span> is inline container, used to mark up a part of a text, or a part of a document.
Flex
  • Either row or column
  • Syntax: selector { display: flex; }
Grid
  • Offers a grid-based layout system
  • Sounds almost the same as flex, but its 2D while flex is 1D.
  • Syntax: selector { display: grid; }
Inline-block
  • Container will be displayed side by side instead of on top of each other.
  • Syntax: selector { display: inline-block; }
