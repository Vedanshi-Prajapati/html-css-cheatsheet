INTRODUCTION TO HTML

What Is HTML ?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages.
  • HTML describes the structure of a Web page
  • It consists of a series of elements that tell the browser how to display the content.
  • A web browser reads the html documents and display them correctly.
  • Web pages can be created and modified by using professional HTML editors.

Basic structure of an HTML document

  1. Doctype declaration
    • "DOCTYPE html" : This declaration defines the document type and indicates to the browser that it's an HTML5 document.
      Sortcut key: ! + tab/enter
  2. HTML root element
    • html: This is the root element that encapsulates all the content on the page, including the head and body elements.
    • It often includes the lang attribute, specifying the document's language for better SEO and accessibility.
  3. Head section
    • head : This section contains metadata about the document that isn't displayed directly on the webpage but is crucial for the page's behavior and rendering.
    • "meta charset="UTF-8": Specifies the character encoding for the document, ensuring proper display of characters from various languages. Emojis are characters from the UTF-8 character set.
    • "meta name="viewport" content="width=device-width, initial-scale=1.0" : Controls the layout and scaling on different devices, especially for mobile responsiveness.
    • "title": Defines the title of the web page, which appears in the browser tab and is also used by search engines for ranking. Search Engine Optimization (SEO)
  4. Body section
    • "body": This section contains all the visible content of the webpage, such as headings, paragraphs, images, links, tables, and more.

    Elements, Attributes and Tags

    Attribute + tags = Elements
    Element is everything from the start tag to the end tag.
    Tag is a keyword in html enclosed in angle brackets. Some HTML elements have no content nor end tag. These elements are called Empty elements.
    Attributes provide additional information about elements, specified in the start tag.

    Nested HTML Elements

    HTML elements can be nested. All HTML documents consist of nested HTML elements.

    HTML Display

    • This is not sure how HTML will be displayed.
    • Large or small screens, and resized windows will create different results.
    • With HTML, you cannot change the display by adding extra spaces or extra lines in your HTML code.
    • The browser will automatically remove any extra spaces and lines when the page is displayed.

    Tag as plain text

    Tags can not be displayed as a plain text directly as the web browser will process that too. Instead use < for < and < for >.
