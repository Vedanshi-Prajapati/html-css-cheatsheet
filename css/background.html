BACKGROUND PROPERTIES

CSS background properties are used to control the background styling of elements on a webpage. The background property is a shorthand property for:
  • background-color
  • background-image
  • background-position
  • background-size
  • background-repeat
  • background-origin
  • background-clip
  • background-attachment
CSS Syntax:
background: bg-color bg-image position/bg-size bg-repeat bg-origin bg-clip bg-attachment initial|inherit;
=> initial :Sets this property to its default value & inherit:Inherits this property from its parent element.
sr Property About Content
1 background-color Sets the background color of an element. Default value is transparent. defined using color names,rgb,hex,hsl
2 background-image Sets one or more images as the background for an element. Supports multiple images, separated by commas, where the first image in the list is closest to the viewer. background-image: url('');
3 background-attachment Determines if the background image scrolls with the page or fixed. fixed, local, scroll, initial, inherit
4 background-clip defines how far the background (color or image) should extend within an element. border-box, padding-box, content-box
5 background-origin specifies the origin position (the background positioning area) of a background image. This property has no effect if background-attachment is "fixed". border-box, padding-box, content-box
6 background-position this property sets the starting position of a background image. It also has background-position-y and background-position-x property. left top, left center, left bottom, right top, right center, right bottom, center top,center center, center bottom (If only one keyword is specified, the other value will be "center"), x% y%, xpos ypos
7 background-repeat Controls how a background image is repeated. repeat, repeat-x, repeat-y, no-repeat, space, round, initial, inherit
8 background-size specifies the size of the background images.one valued-width and two valued-width,height auto, length, percentage, cover, contain
This is an example.
background-color: grey;
background-image: url('');
background-repeat: no-repeat;
background-position: center;
