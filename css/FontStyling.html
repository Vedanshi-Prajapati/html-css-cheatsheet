FONT STYLING

CSS offers various properties to style fonts within web documents. These properties allow control over aspects such as the font's appearance, size, weight, and style.
  • font-family
  • font-size
  • font-weigh
  • font-style
  • line-height
  • font-variant
  • font-stretch
  • Shorthand Property
font-family:
Specifies the typeface to be used. Multiple font names can be provided as a comma-separated list, acting as fallback options if the primary font is unavailable. Generic font families
font-size:
Specifies the size of the font.
