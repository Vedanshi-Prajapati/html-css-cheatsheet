CSS BOX MODEL

The CSS Box Model is a fundamental concept in CSS that describes how every HTML element is rendered as a rectangular box on a webpage. This model defines the dimensions and spacing of elements and is crucial for controlling layout and visual presentation.
The box model consists of four main components, which are:
sr Component Define Property
1 Content Innermost area where the actual content of the element resides, such as text, images, or other media. width, height
2 Padding A transparent space that surrounds the content area. padding-top, padding-right, padding-bottom, padding-left
3 Border A visible line that encloses the padding and content areas. border-width, border-style, border-color
4 Margin A transparent space outside the border area, providing spacing between the element and neighboring elements. margin-top, margin-right, margin-bottom, margin-left
1. Padding, Margin
  • Values: pixels (px), ems (em), rems (rem), percentages (%), or keywords like auto
  • selector{padding-top: value; padding-right: value; padding-bottom: value; padding-left: value;}
  • selector{ padding: value1, value2, value3, value4; } => padding applied in clockwise direction.
  • selector{ padding: value1, value2, value3;} => padding applied in top, right & bottom.
  • selector{ padding: value1, value2;} => padding applied in top-bottom & right-left.
  • ALSO APPLY FOR MARGIN.
  • The auto Value: You can set the margin property to auto to horizontally center the element within its container. The element will then take up the specified width, and the remaining space will be split equally between the left and right margins.
    The inherit Value: This lets the margin of the element be inherited from the parent element.
2. Border
  • Border Style:

    dotted dashed solid double groove ridge inset outset

  • Border Width:
    The border-width property specifies the width of the four borders. The width can be set as a specific size (in px, pt, cm, em, etc) or by using one of the three pre-defined values: thin, medium, or thick.The border-width property can have from one to four values (for the top border, right border, bottom border, and the left border).

    5px thin medium thick

  • Border Color:
    used to set the color of the four borders. If border-color is not set, it inherits the color of the element. border-color property can have from one to four values.

    red blue yellow green

  • Border Sides:
    specifies each of the borders (top, right, bottom, and left). Example: example of border sides
  • Border - Shorthand Property:
    The border property is a shorthand property for the following individual border properties: border-width, border-style (required), border-color.
  • Border-radius: is used to add rounded borders to an element. Example: radius 50px
3. Outline
An outline is a line drawn outside the element's border. An outline is a line that is drawn around elements, OUTSIDE the borders, to make the element "stand out".
  • outline-style, outline-color, outline-width, shorthand=> same as border
  • Outline Offset: Adds transparent space between an outline and the border of an element.

4. Box Shadow
  • box-shadow property is used to apply one or more shadows to an element. In its simplest use, you only specify a horizontal and a vertical shadow. The default color of the shadow is the current text-color.
  • selector{ box-shadow: right, bottom, blur, color, spread, inset; }
  • eg: div { box-shadow: 10px 10px 5px lightblue inset; }
  • An element can also have multiple shadows: div { box-shadow: 5px 5px blue, 10px 10px red, 15px 15px green; }

Box-sizing in CSS :

The CSS box-sizing property determines how the width and height of an element are calculated.
1. content-box (default):
This is the default behavior in CSS. When you set a width and height on an element, these dimensions apply only to the content area. The padding and border are added outside of this specified width and height, causing the element to appear larger than you initially defined.
2. border-box :
With border-box, the specified width and height of an element include the content, padding, and border. This means that if you set an element to be 200px wide, the total width, including any padding and border, will be 200px. The content area will shrink to accommodate the padding and border within the specified dimensions.
