CSS SELECTORS

CSS selectors are used to "find" (or select) the HTML elements you want to style. A CSS selector selects the HTML element(s) you want to style.

    1. Universal Selector
    Selects all elements on the page and applies the same style universally.
    Example: * { color: red; }
    Here, the h1 is the selector that selects all the h1 elements of our document and changes their color to red.
    2. Element Selector
    The element selector selects HTML elements (p, div, h1, etc) and applies CSS to them.
    example: h1 { color: red; } p { color: orange; }
    In the above example, the element selector h1 selects all h1 elements and styles their color to red p selects all p elements and styles their color to orange
    3. Class Selector
    The class selector selects the HTML element using the class attribute and applies CSS to it. The class selector is specified using the period (.) character, followed by the class name.
    Example: .first-paragraph { background-color: orange; }
    The class selector .first-paragraph selects all the paragraphs having the first-paragraph class name and styles background-color to orange.
    4. ID Selector
    The id selector selects the HTML element with a unique identifier (id) and adds CSS to it.The id selector is specified using the hash (#) character, followed by the id of the element. The id selector is unique and selects one unique element.
    Example: #second-paragraph { color: red; }
    The id selector #second-paragraph selects the second paragraph and styles the text color to red.
    5. Group Selector
    The group selector allows you to select multiple elements and apply the same style to all of them.
    Example: h1, p { color: blue; }
    Here, the code applies CSS styling to all h1 and p elements. Notice that we have used , to separate the HTML elements..
    6. Descendant Selector ( )
    The CSS descendant selector targets elements that are nested anywhere within another specified element in the HTML document structure. It is represented by a single space character between two or more selectors.
    Example: .container p { color: blue; }
    In this example, both "Paragraph 1" and "Paragraph 2" would have their text color set to blue because they are both descendants of the div with the class container. "Paragraph 3" would not be affected as it is not a descendant of .container.
    7. Child Selector (>)
    The child selector in CSS, denoted by the > symbol, is a combinator used to select elements that are the direct children of a specified parent element. It establishes a direct parent-child relationship in the HTML document structure.
    Example: .container > p { color: blue; }
    In this example, only the first paragraph "This is a direct child paragraph." would turn blue, as it is a direct child of the div.container. The second paragraph, being a child of the section element (which is itself a child of div.container), would not be affected by this specific rule because it is not a direct child of .container.
    8. Adjacent Sibling Selector
    The adjacent sibling selector in CSS, denoted by a plus sign (+), selects an element that immediately follows another specified element, provided both elements share the same parent.
    Example: h2 + p { color: blue; }
    This would only apply the blue color to the first "p" element that appears directly after an "h2" element, and not to any other "p" elements, even if they are also siblings of the h2 but not immediately adjacent.
    9. General Sibling Selector
    The general sibling selector in CSS, represented by the tilde symbol (~), selects all sibling elements that follow a specified element and share the same parent.
    Example: h1 ~ p { color: blue; }
    This CSS rule would style both "Paragraph 1" and "Paragraph 2" in blue, as they are both "p" elements that are siblings of "h1" and appear after it within the same "div> parent. The 'span' element would not be affected as it is not a "p" element.
    10. Attribute Selector
    CSS attribute selectors allow targeting HTML elements based on the presence or value of their attributes. CSS attribute selectors target HTML elements based on the presence or value of their attributes. This provides a powerful way to style elements more precisely than with just class or ID selectors.
    Example: div[target] { background-color: yellow; }
    his rule would apply a yellow background to any
    element that has a target attribute. Exact attribute value match.
    11. Pseudo-class
    A CSS pseudo-class is a keyword added to a selector that defines a special state of the selected element(s). It allows for styling elements based on their state or position in the document tree without adding extra classes to the HTML. A pseudo-class is a keyword added to a CSS selector, prefixed by a colon (:), to define a specific state or condition of an element. It is used to style elements like a hovered button, the first child of a container, or checked input fields.
    Example: button:hover { background-color: lightblue; color: white; }
    12. Pseudo-element
    A pseudo-element is a keyword added to a selector that lets you style specific parts of an element. For example, you can style the first line of a paragraph, add content before or after an element, or create complex effects with minimal code. Pseudo-elements are denoted by a double colon (::) (or : in legacy CSS).
    Example: p::before { content: " "; color: gold; }
