DISPLAY PROPERTY IN CSS

The CSS display property controls the layout and rendering behavior of an HTML element within the document flow. It determines how an element generates its box and interacts with other elements on the page.
Common display values and their effects:
1. Block
  • lock take up the full available width of their parent container.
  • They force a line break before and after the element, meaning other elements will be placed above or below them, not beside them.
  • hey accept height and width properties.
2. Inline
  • Elements with display: inline do not start on a new line and only occupy the necessary width and height based on their content.
  • They do not accept width and height properties.
  • They only respect horizontal margins and padding, not vertical.
3. Inline-block
  • This value combines characteristics of both inline and block elements.
  • inline-block flow like inline elements (sitting next to each other) but they also allow you to set width, height, margins (all directions), and padding.
4. None
  • none are completely hidden from the page and do not take up any space in the document flow.
  • This is often used with JavaScript to dynamically show or hide elements.
5. Flex:
  • This value turns an element into a flex container, allowing for flexible and responsive layouts.
  • Child elements (flex items) within the container can be aligned, distributed, and resized efficiently.
  • Flexbox is primarily used for one-dimensional layouts (rows or columns).
6. Grid
  • This value transforms an element into a grid container, enabling a two-dimensional grid system.
  • It allows for the creation of layouts with rows and columns, providing precise placement of child elements.
