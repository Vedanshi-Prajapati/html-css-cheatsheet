TRANSFORM PROPERTY IN CSS

The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements. The transformations can be of 2-D or 3-D type.
Syntax:
transform: none|transform-functions|initial|inherit;

2-D Transforms Functions

1. matrix
combines all the 2D transform functions into one.
The parameters are as follow: matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())
2. rotate
rotates an element clockwise or counter-clockwise according to a given degree. Using negative values will rotate the element counter-clockwise.
3. translate
Moves an element along the X and/or Y axis.
4. scale
increases or decreases the size of an element according to the parameters given
5. scaleX
ncreases or decreases the width of an element.
6. scaleY
increases or decreases the height of an element.
7. skewX
skews an element along the X-axis by the given angle.
8. skewY
skews an element along the Y-axis by the given angle.
9. skew
skews an element along the X and Y-axis by the given angles.


3-D Transforms

CSS 3D transforms enable the manipulation of HTML elements within a three-dimensional space, allowing for effects like rotation, translation, and scaling along the X, Y, and Z axes.
transform-style: preserve-3d: This allows child elements to retain their 3D positioning when their parent element has a 3D transform applied, creating a unified 3D space shared by multiple elements.
Transformation methods are:
1. translate3d(x, y, z)
Moves an element along the x, y, and z axes.
2. rotateX(angle)
Rotates an element around its horizontal x-axis.
3. rotateY(angle)
Rotates an element around its vertical y-axis.
4. rotateZ(angle)
Rotates an element around its z-axis, perpendicular to the screen.
5. rotate3d(x, y, z, angle)
Rotates an element around an axis defined by the x, y, and z vector, at a specified angle.
6. scale3d(x, y, z)
Scales an element along the x, y, and z axes, changing its width, height, and depth.
7. perspective()
Defines a perspective view for a 3D transformed element

TRANSITIONS IN CSS

CSS transitions are used to create smooth animations between two states of an element, enhancing interactivity and user experience.
Always specify the transition-duration property, otherwise the duration is 0, and the transition will have no effect.
Transition Properties are:
1. transition-delay
specifies when the transition effect will start, is defined in seconds (s) or milliseconds (ms).
2. transition-property
specifies the name of the CSS property the transition effect is for (the transition effect will start when the specified CSS property changes).
3. transition-duration
specifies how many seconds (s) or milliseconds (ms) a transition effect takes to complete.
4. transition-timing-function
specifies the speed curve of the transition effect.
  • ease
  • linear
  • ease-in
  • ease-out
  • ease-in-out
  • step-start
  • step-end
  • steps(int,start|end)
  • cubic-bezier(n,n,n,n)
  • initial/inherit
5. Shorthand Property
The four individual transition properties can be combined into a single shorthand property: transition. The order of values matters within the shorthand.
transition: ;
